# How to Run This Project

This project has two main parts:

```text
Python ranking pipeline
React frontend dashboard
```

The Python pipeline builds the cleaned datasets, ranking scores, and skill graph.
The React frontend visualizes candidate fit with a radar chart and skill graph.

---

## Important Files

### Python pipeline

```text
preprocess_tech.py
build_vectors_and_graph.py
analyze_graph.py
data_profile.py
skill_taxonomy.json
```

Purpose:

```text
preprocess_tech.py
    Filters tech roles, cleans skills, normalizes aliases, and creates cleaned CSV files.

build_vectors_and_graph.py
    Builds the skill graph, computes ranking scores, and exports ranked results.

analyze_graph.py
    Prints graph quality stats like node count, edge count, density, and strongest edges.

data_profile.py
    Profiles the raw datasets and writes preview/report files.

skill_taxonomy.json
    Defines technical skill domains used by the graph and scoring logic.
```

### Input data files

```text
resume_data_for_ranking.csv
job_skill_set.csv
```

These are the raw datasets used by the Python pipeline.

### Generated output files

```text
clean_tech_resume_dataset.csv
clean_tech_job_skill_dataset.csv
ranked_tech_resumes.csv
skill_graph.json
skill_vocab_final.json
initial_skill_vocab.json
skill_frequency.json
graph_summary.txt
dataset_analysis.txt
```

These are regenerated by the Python scripts.

### Frontend files

```text
frontend/
frontend/src/main.jsx
frontend/src/styles.css
frontend/scripts/sync-data.js
frontend/public/data/
```

Purpose:

```text
frontend/src/main.jsx
    Main React dashboard with Radar and Skill Graph views.

frontend/src/styles.css
    Dashboard styling.

frontend/scripts/sync-data.js
    Copies generated Python outputs into the frontend public data folder.

frontend/public/data/
    Data files served to the browser by Vite.
```

# Run the Python Pipeline

From the project root:

```powershell
cd "C:\Users\Dell\OneDrive\Desktop\resume ranker"
```

Run preprocessing:

```powershell
venv\Scripts\python.exe preprocess_tech.py
```

Build vectors, ranking scores, and graph:

```powershell
venv\Scripts\python.exe build_vectors_and_graph.py
```

Analyze graph quality:

```powershell
venv\Scripts\python.exe analyze_graph.py
```

Optionally profile the raw datasets:

```powershell
venv\Scripts\python.exe data_profile.py
```

Expected important outputs:

```text
ranked_tech_resumes.csv
skill_graph.json
graph_summary.txt
```

# Run the Frontend Dashboard

Go to the frontend folder:

```powershell
cd "C:\Users\Dell\OneDrive\Desktop\resume ranker\frontend"
```

Install dependencies:

```powershell
npm install
```

Copy the latest generated pipeline outputs into the frontend:

```powershell
npm run sync-data
```

Start the local dev server:

```powershell
npm run dev -- --host 127.0.0.1 --port 5173
```

Open:

```text
http://127.0.0.1:5173/
```

The frontend currently has:

```text
Radar view
    Recruiter-facing candidate fit radar chart.

Skill Graph view
    Interactive Cytoscape skill graph using skill_graph.json.
```

Build the frontend for production:

```powershell
npm run build
```
